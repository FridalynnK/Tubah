# 🌿 Tubah Subdivision — Community Information Hub

> A civic information website tracking the health, agriculture, land affairs, education, and developmental progress of **Tubah Subdivision**, Mezam Division, North West Region, Republic of Cameroon.

---

## 📌 Overview

This project is a single-page HTML website that serves as a public civic resource for residents, administrators, and visitors of Tubah Subdivision. It brings together key sectoral information in one accessible place and allows community members to submit observations directly to the subdivision administration.

---

## 🗂️ Sections

| Section | Description |
|---|---|
| **Hero** | Introduction and quick navigation to key sections |
| **Stats Band** | At-a-glance figures — population, health facilities, projects, land disputes |
| **❤️ Health** | Overview of health facilities, prevalent conditions, maternal health, and drug supply |
| **🌿 Agriculture** | Key crops, livestock, extension services, and sector coverage estimates |
| **⚖️ Land Disputes** | Categories of land conflicts and resolution pathways |
| **🏗️ Projects** | Ongoing, planned, and stalled infrastructure projects with progress indicators |
| **🎓 Education** | Profiles of the University of Bamenda (Bambili) and National Polytechnic Institute (Bambui) |
| **💬 Observations** | Community form to submit reports to subdivision administration via email |

---

## 🏛️ Featured Institutions

### University of Bamenda (UBa)
- **Location:** Bambili, Tubah Subdivision
- **Founded:** 2010
- **Type:** State University
- Faculties include Science, Arts, Social Sciences, Health Sciences, and the Higher Technical Teacher Training College

### National Polytechnic Institute Bamenda (POLYBAM)
- **Location:** Bambui, Tubah Subdivision
- **Type:** Public technical & vocational institution under UBa
- Departments include Civil Engineering, Electrical Engineering, Computer Science, Mechanical Engineering, and Business & Management

---

## 🛠️ Tech Stack

- **HTML5** — single-file structure, no build tools required
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — scroll reveal, form handling, hamburger menu
- **Google Fonts** — Playfair Display + DM Sans
- No frameworks, no dependencies — fully self-contained

---

## 📱 Responsive Design

The site is fully responsive:
- **Desktop** — horizontal navigation bar with all sector links
- **Mobile (≤768px)** — collapsible hamburger menu with animated drawer, icon-labelled sector links, and single-column layouts

---

## 📨 Community Observations

The observation form at the bottom of the page allows residents to submit reports on the following categories:
- Health
- Agriculture
- Education
- Land Dispute
- Road Construction
- Water & Electricity
- Other

Submissions are sent via `mailto:` to the subdivision administration email.

## 🚀 Getting Started

No installation needed. Simply open `index.html` (or `tubah.html`) in any modern browser.

```bash

git clone https://github.com/FridalynnK/tubah.git

# Open in browser
open index.html
```
